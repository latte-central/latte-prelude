eq-trans*

Transitivity of equal, a n-ary version of eq-trans. The parameter eq-terms is a vector of at least two equalities.

For example:

``` (eq-trans* eq1 eq2) ==> (eq-trans eq1 eq2)

(eq-trans* eq1 eq2 eq3) ==> (eq-trans (eq-trans eq1 eq2) eq3)

(eq-trans* eq1 eq2 eq3 eq4) ==> (eq-trans (eq-trans (eq-trans eq1 eq2) eq3) eq4) ``` etc.

equality

(equality [T :type] [x T] [y T])
(forall [P (==> T :type)] (<=> (P x) (P y)))

Definition: The intuitionistic, second-order definition of equality. This corresponds to Leibniz’s indiscernibility of identicals.
